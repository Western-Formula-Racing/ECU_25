\doxysubsection{Detailed Description}
Main entry point for the ECU firmware. 

\begin{DoxyAuthor}{Author}
Yamaan Bakir
\end{DoxyAuthor}
This file contains the Free\+RTOS app\+\_\+main() function which initializes the \doxylink{class_c_a_n}{CAN} interface, selects either the front or rear ECU mode based on a GPIO pin, and continuously monitors the \doxylink{class_c_a_n}{CAN} bus state for errors. It sets up the required tasks (State\+Machine or Rear\+ECU) accordingly and handles TWAI bus recovery and logging. \begin{DoxyVersion}{Version}
