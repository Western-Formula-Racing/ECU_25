\doxysubsection{Detailed Description}
Represents a single \doxylink{class_c_a_n}{CAN} signal, including its position, length, scaling, and value. 

This class provides methods to encode and decode individual signal values from \doxylink{class_c_a_n}{CAN} messages. It supports automatic scaling, endian handling, and timeout logic for stale signals.
