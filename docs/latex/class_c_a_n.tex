\doxysubsection{Detailed Description}
Manages \doxylink{class_c_a_n}{CAN} bus communication using the ESP32 TWAI driver. 

This class handles \doxylink{class_c_a_n}{CAN} initialization, reception, and periodic transmission of \doxylink{class_c_a_n}{CAN} messages. It supports configurable TX/\+RX pins and mode, signal-\/level mapping, and optional logging of messages for diagnostics or telemetry.

Periodically transmits \doxylink{class_c_a_n}{CAN} messages at 10ms, 100ms, and 1000ms intervals. 

Triggered every 10ms by a timer, this function both builds and transmits \doxylink{class_c_a_n}{CAN} messages. It iterates over preset ids and signals with the help of the various other \doxylink{class_c_a_n}{CAN} classes, each message is then populated using the latest data from CAN\+\_\+\+Map and sent over the network. If logging is enabled then this function also logs the messages.
