Detailed Description

Main entry point for the ECU firmware.

Author
Yamaan Bakir

This file contains the FreeRTOS app_main() function which initializes the CAN interface, selects either the front or rear ECU mode based on a GPIO pin, and continuously monitors the CAN bus state for errors. It sets up the required tasks (StateMachine or RearECU) accordingly and handles TWAI bus recovery and logging.
